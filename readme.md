# Календарь праздников 2025

Получение списка праздников в России на 2025 год через Holiday API.

## Как установить

1. Зарегистрируйтесь на [Calendarific](https://calendarific.com) и получите свой API ключ
2. Создайте файл `.env` в корневой папке проекта
3. Добавьте в `.env` ваш ключ:

API_KEY=ваш_секретный_ключ

Python3 должен быть уже установлен. Затем используйте pip для установки зависимостей:

pip install -r requirements.txt

## Как запустить проект

Откройте командную строку (cmd) в папке с проектом и выполните:

python main.py

## Цель проекта

Код написан в образовательных целях на онлайн-курсе для веб-разработчиков.
